=====================================================
  COMO USAR OS DIAGRAMAS NO MERMAID.LIVE
=====================================================

1. Acesse: https://mermaid.live

2. Apague o codigo de exemplo que aparece na esquerda

3. Abra UM dos arquivos .mermaid desta pasta

4. Selecione TUDO (Ctrl+A) e copie (Ctrl+C)

5. Cole no mermaid.live (Ctrl+V)

6. O diagrama aparece na direita automaticamente

7. Para exportar: clique "Actions" > "PNG" ou "SVG"


=====================================================
  ARQUIVOS DISPONIVEIS
=====================================================

01_visao_geral.mermaid
   -> Arquitetura completa do sistema
   -> Browser + Next.js + Supabase + Shopify

02_modelo_dados.mermaid  
   -> Tabelas do banco de dados
   -> members, referral_events, shopify_customers, roles

03_fluxo_cadastro.mermaid
   -> Passo a passo do cadastro de um membro
   -> Diagrama de sequencia

04_tags_shopify.mermaid
   -> Como os dados viram tags no Shopify
   -> lrp_member, lrp_ref, lrp_sponsor, lrp_status

05_resumo_executivo.mermaid  [RECOMENDADO PARA CLIENTE]
   -> Versao simplificada em 4 passos
   -> Colorido e facil de entender
