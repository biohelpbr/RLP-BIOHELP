# F-V22 — Avisos no painel (announcement bar CMS)

Barra de aviso no topo do dashboard do membro (`V2Dashboard`), gerenciada por CRUD admin em `/admin/announcements` (mensagem + upload de imagem + link/CTA + cor + janela de datas + on/off). Só o aviso ativo mais recente dentro da janela aparece (`getActiveAnnouncement`); não-fechável. Tabela `announcements` + bucket público `announcements`. Banner responsivo por aspect-ratio (mobile mais alto, desktop mais baixo, sem cortar o título).
