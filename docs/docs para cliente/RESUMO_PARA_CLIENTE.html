Biohelp LRP Pivô V2 em migração

Resumo Executivo · Atualizado em 29/04/2026 · Deadline da migração: 11/05/2026

📣
Modelo do programa foi redefinido.

O modelo MLM/CV original (Sprints 1-7, 98% das funcionalidades v1 entregues) foi descontinuado pela direção da Biohelp em 28/04/2026, baseado no novo fluxograma e nos comentários no escopo. O novo modelo é afiliação de 1 nível com promoção a Founder ao atingir 5 ativos no clube, comissão 50% por convite, créditos Shopify pré-Founder e saque cash via NF de serviço.

Prazo: migração completa até 11/05/2026 (13 dias corridos / 8 dias úteis a partir de hoje).

🔄 O que mudou no projeto

❌ REMOVIDO do modelo

  • CV (Commissionable Value)
  • Múltiplos níveis (N0/N1/N2)
  • Rankings Parceira/Líder/Diretora/Head
  • Status mensal por 200 CV
  • Reset mensal de CV
  • Compressão de rede após 6 meses
  • Fast-Track 30%/20%
  • Comissão Perpétua diferenciada
  • Bônus 1, 2, 3
  • Leadership Bônus
  • Royalty
  • RPA / CPF / limite R$1.000
  • House Account (cadastro sem ref)

🔄 ALTERADO (mantém com regra nova)

  • Cadastro: ref obrigatório (link OU código). Sem ref → bloqueia
  • Status ativo: assinatura paga (não mais CV)
  • Visão da rede: só sponsor + indicados diretos (1 nível)
  • ref_code: mantém formato BH00001 + opção de digitar código manual
  • Pagamento: Cashin (provável) ou crédito em loja. CPF e CNPJ ambos podem receber
  • Sync Shopify: tags + Locksmith continuam liberando preço de clube
  • Cupom de creatina: vira sistema de campanhas (admin define quem/quando)

➕ NOVO no V2

  • Integração com Guru (assinatura)
  • Comissão 50% direta sobre assinatura do convidado
  • Saldo + créditos Shopify 1:1
  • Promoção a Founder ≥5 ativos no clube
  • Saque cash via Cashin + NF de serviço (Founder com CNPJ)
  • Ranking de Founders
  • Área de conteúdo (texto/vídeo/PDF)
  • Link de WhatsApp por Founder
