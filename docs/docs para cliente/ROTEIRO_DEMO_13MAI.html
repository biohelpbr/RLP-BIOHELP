4

Resultado & Resgate · Tier de comissão

localhost:3000/dashboard/finance

Mostre os 3 cards de saldo no topo (Disponível, Pendente Net-15, Recebido total) e o card “Sua taxa de comissão” com a tabela de 4 tiers.

  • Tabela de tiers (40/45/50/55%) — regra exata a definir com cliente
  • Cálculo de tier por afiliadas ativas (server-side)
  • Imposto 15% deduzido sempre
  • Saldo R$ 290,40 vem do commission_ledger v1

Como vocês decidiram que a comissão é variável por nº de afiliadas ativas — coloquei aqui 4 tiers escalonados (40% → 55%) e o sistema já calcula automaticamente em qual a parceira está. Os percentuais exatos são placeholder até vocês fecharem a regra final, mas o motor está pronto pra qualquer escala. Imposto 15% já desconta sempre.

5

Solicitar resgate · Triple-resgate

/dashboard/finance → botão “Solicitar resgate”

Abra o dialog. Mostre as 3 abas: PIX (CNPJ + NF), Cashback Cashin, Crédito Shopify.

Em cada aba, aponte o breakdown:

  • Bruto (valor solicitado)
  • Imposto (15%) — sempre aparece, valor calculado em tempo real
  • Taxa PIX — só aparece se aba PIX (R$ 3,67 fixos)
  • Você recebe — líquido final

Em PIX: mostre os dados de NF Biohelp com o badge “a confirmar com a Biohelp”.

  • 3 métodos com Server Action
  • Breakdown matemático correto em tempo real
  • Razão Social/CNPJ/Endereço/IE — placeholders inventados
  • Cliente ainda não enviou dados reais da NF

Vocês decidiram que vão sempre descontar ~15% de imposto, em qualquer dos 3 métodos. Aqui na UI o membro vê isso explicitamente — ‘bruto X, imposto Y, líquido Z’. Sem surpresa pós-aprovação. Os dados de NF (essa caixa amarela ⚠️) estão como placeholder esperando vocês passarem os reais — Razão Social, CNPJ, Endereço e IE — antes do go-live de 11/06.

6

Demonstrar resgate Crédito Shopify (sem NF)

Selecione aba “Crédito Shopify”, troque o valor pra 50,00. Mostre o breakdown automático: Bruto R$ 50,00 → Imposto -R$ 7,50 → Líquido R$ 42,50.

Clique “Solicitar”. Toast verde “Crédito solicitado!” aparece. Saldo Disponível desce de R$ 290,40 para R$ 240,40.

  • Grava na tabela payout_requests (status=pending)
  • tax_amount + net_amount gravados
  • Saldo recalculado via RPC get_available_balance

Resgates — moderar o pedido da parceira

/admin/payouts

Abra a aba “Crédito Shopify”. Você vai ver o pedido R$ 50,00 / Líquido R$ 42,50 da Sponsor Teste em status “Pendente”.

Clique Aprovar → status muda pra “Aprovado”. Botões viram “Marcar pago” / “Rejeitar”. Clique Marcar pago → status “Pago”.

  • Server Action approvePayout / markPayoutPaid
  • Estados terminais explícitos (não pode aprovar 2x)

Aqui vocês moderam todos os pedidos — vê quem pediu, quanto bruto, quanto líquido depois do imposto. Aprovar libera, “Marcar pago” é depois de efetivar o crédito na Shopify / PIX / Cashin. Tudo registrado.
